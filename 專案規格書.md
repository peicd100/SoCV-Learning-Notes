# 專案規格書

## 基本資訊
- 專案名稱：`university notes`
- 專案類型：MkDocs 靜態網站 / Python 工具專案
- conda 環境：`mkdocs`
- 專案根目錄：`y:\github_note\university notes`
- 正式站點 URL：`https://peicd100.github.io/university-notes/`

## 專案用途
整理大學課程筆記與 blog 內容，使用 MkDocs Material 產出靜態網站，並保留自訂主題、搜尋、RSS、加密頁面與數學公式支援。

## 主要技術
- Python 3.11
- MkDocs
- MkDocs Material
- MkDocs RSS / Tags / Encrypt Content / Offline
- 自訂主題覆寫：`docs/theme/`

## 目錄說明
- `docs/`：Markdown 原始內容、logo、theme 覆寫
- `docs/theme/`：Material 主題覆寫與自訂 CSS / HTML
- `site/`：MkDocs 建置輸出
- `tools/project_launcher.py`：啟動本機靜態站的 Python 啟動器
- `tools/source_jump_hook.py`：`mkdocs serve` 本機專用的原文定位 API
- `vbs_bat/run.bat`：隱藏啟動流程的批次腳本
- `vbs_bat/university notes.vbs`：不顯示 bat 視窗的入口
- `mkdocs.yml`：網站主設定

## 執行流程
1. 開發預覽：`Y:\conda\envs\mkdocs\python.exe -m mkdocs serve --dirty --livereload`
2. 建置輸出：`Y:\conda\envs\mkdocs\python.exe -m mkdocs build --clean`
3. 隱藏啟動：執行 `vbs_bat\university notes.vbs`
4. 打包前置：先 `mkdocs build --clean`
5. 打包啟動器：使用 `PyInstaller` 打包 `tools\project_launcher.py`
6. 選字右鍵開原文檔案：僅在 `mkdocs serve` 預覽時可用，因為功能依賴 serve 模式下新增的本機 API，並會由本機直接呼叫 VS Code CLI 開檔

## 目前重要設定
- 主題：`material`
- 自訂主題目錄：`docs/theme`
- 主要首頁入口：`blog/index.md`
- 額外樣式：`assets/pymdownx-extras/extra-8611f6c398.css`、`assets/pymdownx-extras/自定義.css`
- 額外腳本：`assets/pymdownx-extras/extra-loader-iLQ-keay.js`、`assets/pymdownx-extras/toc-fold.js`
- 已啟用功能：navigation tabs、footer、instant、tracking、top、toc.follow、code copy、search.share

## 啟動器規格
- `run.bat` 會先用 `mkdocs build --clean` 更新 `site/`
- 建置成功後，以 `tools\project_launcher.py` 啟動本機 HTTP 服務
- 啟動器預設優先開啟 `blog/index.html`
- 打包時使用 `--add-data "site;site"` 將靜態網站併入可執行檔
